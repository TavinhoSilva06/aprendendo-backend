Nest (NestJS) é uma estrutura para a construção de aplicativos Node.js do lado do servidor eficientes e escalonáveis. 
Ele usa JavaScript progressivo, é construído e oferece suporte total a TypeScript (mas ainda permite que os desenvolvedores 
codifiquem em JavaScript puro) e combina elementos de OOP (Programação Orientada a Objetos), FP (Programação Funcional) e 
FRP (Programação Funcional Reativa).

TypeScript: O TypeScript é uma linguagem criada em cima do JS e permite que adicionemos tipagem de dados
em nossas variáveis, além de nos oferecer recursos avançados, como a Orientação a Objetos e padrões de projeto.

Os tipos de dados especificam o que podemos colocar nos nossos objetos do banco de dados, podendo ser tabelas,
stored procedures, variáveis e etc. integridade do banco de dados depende decisivamente dos tipos de dados com escopo apropriado
